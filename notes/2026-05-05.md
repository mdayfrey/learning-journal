# 2026-05-05

Learn Git Branching — Introduction Sequence + Ramping Up
- HEAD — pointer to current commit you're working on
- Detached HEAD — HEAD points directly to a commit instead of a branch
- Relative refs — navigate commits with `^` (one up) and `~n` (n steps up)
- `git branch -f main HEAD~3` — force move a branch to a different commit
- `git reset HEAD~1` — undo commit locally (rewrites history)
- `git revert HEAD` — undo commit safely by adding a new commit (safe for remote)
