Shell tools, redirection, pipes
- Redirection: `>` (overwrite), `>>` (append), `2>` (stderr), `&>` (both streams)
- Pipes: `|` chains commands together — stdout of one becomes stdin of next
- `grep` — filters output by pattern
- `head` / `tail` — first/last lines of a file
- `cat` — concatenate files or view contents
- `less` — view long output interactively (spacebar to scroll, q to quit)
- `ps aux | grep <process>` — find running processes

Git collaboration + branching
- `git pull` — sync latest changes from GitHub
- `git clone` — copy a public repo locally
- `git log --oneline` — view condensed commit history
- `git checkout -b <branch>` — create and switch to new branch
- `git checkout main` — switch back to main
- `git merge <branch>` — bring branch changes into main
- Conventional commits: feat / fix / docs / chore / refactor / test / style
